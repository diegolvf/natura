Modelagem no Solver

Utilizamos o Solver do Excel para resolver o problema de programação linear, considerando as restrições de tempo e matéria-prima, com o objetivo de maximizar o lucro total.

O modelo definiu as variáveis de decisão como o número de unidades produzidas de cada produto, respeitando as restrições de produção.
